Приложение для поддержки ментального здоровья с использованием методов КПТ (когнитивно-поведенческая терапия).

## 📱 Возможности

- 📝 **Дневник мыслей** - запись и анализ автоматических мыслей
- 🧘 **Упражнения КПТ** - дыхательные практики, работа с мыслями, релаксация
- 📊 **История записей** - просмотр прогресса
- 🏆 **Достижения** - система мотивации
- 🤖 **Чат с ботом** - эмоциональная поддержка 24/7
- 🧬 **ДНК профиль** - глубокий анализ ментального здоровья
- 🎵 **Музыка** - расслабляющие звуки
- 🌬️ **Дыхательные упражнения** - интерактивный таймер

## 🚀 Установка

### Требования
- Python 3.8+
- pip

### Установка зависимостей

```bash
pip install -r requirements.txt
```
### Запуск

```bash
python main.py
```

### Технологии
- **Python** 3.11

- **PyQt5** - графический интерфейс

- **SQLite** - база данных

- **Matplotlib** - графики

- **scikit-learn** - машинное обучение

- **TensorFlow** - нейросети

- **Pygame** - аудио
